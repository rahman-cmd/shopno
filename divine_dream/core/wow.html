WOW এর Amazing নেটওয়ার্কে আপনাকে স্বাগতম!



WOW হলো একটি স্বাধীন ব্রেইন-টু-ব্রেইন Super PsycoTechnological কমিউনিকেশন নেটওয়ার্ক; যেখানে আপনি প্রাপক (Recipient)- কে কোনরূপ Text / Photo / Voice / Video / Animation / Emoji ইত্যাদির সহায়তা ছাড়াই সম্পূর্ণ Offline- এ আপনার মনের ভাব তথা আপনার ব্রেইনের অনুভূতি Transfer করতে সক্ষম হবেন। WOW এমনই এক PsycoTechnological Communication Network (Social Network) এর প্রারম্ভিক প্রেষণা যেখানে আপনি আপনার পরিচিত প্রাপক ব্যতিরেকেও অপরিচিত [যার চেহারা-অবয়ব আপনি ইতিপূর্বে কখনো দেখেননি] প্রাপকের নিকট আপনার ব্রেইনের অনুভূতি (Mind Message) পাঠাতে সক্ষম হবেন!

Mind Message হলো কোন নির্দিষ্ট বিষয়ে আমাদের ব্রেইনের পরিপূর্ণ Pure অনুভূতি [আমরা আমাদের মনের ভাব প্রকাশে নন-ভার্বাল কমিউনিকেশন, মৌখিক ভাষা, লেখনী ইত্যাদি ব্যবহার করে থাকি; অনলাইন কমিউনিকেশনেও মনের ভাব আদান প্রদানে শুধুমাত্র Text / Photo / Voice / Video / Animation / Emoji এর মাঝেই আমরা সীমাবদ্ধ! বাস্তবিকপক্ষে সত্যিই কি আমরা আমাদের মনের অনুভূতি সম্পূর্ণভাবে অন্যকে রিয়েলাইজ তথা উপলব্ধি করাতে সক্ষম হই? বস্তুত প্রায়শই আমরা আমাদের মনের Exact অনুভূতি যা ঐ মুহূর্তে আমাদের ব্রেইনে বিদ্যমান তা সম্পূর্ণরূপে Purely অন্যকে বোঝাতে সক্ষম হইনা! Mind Message হলো আপনার মনের সেই অনুভূতি বা Brain Data যা দ্বারা আপনি সম্পূর্ণভাবে আপন মনের ভাব অন্যকে রিয়েলাইজ করাতে সক্ষম হউন]।

আসুন ধাপে ধাপে ব্রেইন-টু-ব্রেইন কমিউনিকেশন শেখা যাক:-

সর্বপ্রথম আপনাকে মনে প্রাণে বিশ্বাস আনয়ন করতে হবে যে "ব্রেইন-টু-ব্রেইন কমিউনিকেশন অবশ্যই সম্ভব" কেননা আপনার দৃঢ়চেতা বিশ্বাস (Faith) আপনার ব্রেইনে হয়তো Positive Potential Effect তৈরী করবে এখন আপনি যেমন ফিজিক্যাল অবস্থানে আছেন তা হতে Low Frequentative ফিজিক্যাল অবস্থানে আসুন [মানব শরীর দাড়ানো অবস্থায় স্বাভাবিকভাবে প্রায় 7.5Hz ফ্রিকোয়েন্সি Emit করে, অন্যদিকে বসার ধরন বা ভঙ্গি অনুযায়ী 4Hz থেকে 6Hz ফ্রিকোয়েন্সি নির্গত হয়] "জীবনের সবকিছুই কম্পন" এই Sentence- টি ফিজিক্স'কে সমীহ পূর্বক আপাত ফিলোসফিক ভাবার্থে এমনভাবে ভাবুন "আপনার Brain Message এর নিজস্ব ফ্রিকোয়েন্সি বিদ্যমান" আছে [বাস্তবিক অর্থেই আমাদের প্রতিটি চিন্তা ও আবেগের নিজস্ব কম্পন বা ফ্রিকোয়েন্সি তরঙ্গ রয়েছে] এখন একটি উন্নত সাউন্ড কোয়ালিটির ইয়ারফোন / ইয়ারপড / হেডফোন ব্যবহার করে 33.3Hz এর Sine ওয়েভেশুনতে শুনতে চোখ বন্ধ করে আপন মনে আপনার Brain Message নিয়ে ভাবতে থাকুন; খুব ধীর-স্থির শান্ত চিত্তে আপনার মনোভব'কে যথাসম্ভব প্রখর এবং জীবন্ত করে তুলুন [কল্পনার মাধ্যমে এমনভাবে আপনার Mind Message- কে পরিস্ফুটিত করুন যেন তা আলোক প্যাকেজে ফোটন কণার মতোই কম্পিত হচ্ছে] এখন আপনি চোখ খুলেএর দিকে তাকিয়ে ভাবুন যেন ঘোড়াটি আপনার Mind Message আপনার প্রাপকের নিকট Spectrum আকারে পৌছে দিচ্ছে [এখানে প্রাপক যদি আপনার পরিচিত হয় এবং তিনিও আপনার Mind Message গ্রহণে ইচ্ছুক হন তবে পূর্ব হতেই টাইম শিডিউল এবং মেন্টাল স্টেটমেন্ট সেটআপ ও প্রোগ্রাম করে নিতে পারেন; এক্ষেত্রে Mind Message তরঙ্গ আকারে পাঠানোর সময় উক্ত প্রাপকের মুখচ্ছবি কল্পনা করে নিবেন। অপরাপর প্রাপক যদি আপনার পরিচিত হয় কিন্ত Mind Message গ্রহনে পূর্ব হতেই প্রোগ্রাম/সেটআপ করা সম্ভবপর না হয় তবে স্রেফ তার মুখচ্ছবি ও সহজাত Behavior নিয়ে কল্পনা করবেন। আবার যদি প্রাপক একান্তই আপনার অপরিচিত হয় তবে তার সম্পর্কে আপনার Gather করা তথ্য ও ধারণা প্রয়োগ করে কাল্পনায় একটি অবয়ব/প্রতিকৃতি তৈরী করে নিবেন - এইরূপ একইভাবে আপনি একক ব্যক্তি হতে Mass People এর মাঝেও আপনার Mind Message পৌছে দেওয়ার প্রচেষ্টা চালাতে পারেন] এবার- এ আপনার কল্পনার প্রাপক'কে নিয়ে এমনভাবে ভাবতে থাকুন যেন আপনার Brain Message আপনার প্রাপকের ব্রেইনে পৌছে তাকে কম্পিত করছে [বিষয়টা অনুধাবন করতে DyedGirl এর ওপর Click করলেই আপনি যথার্থরূপে রিয়েলাইজ করতে সক্ষম হবেন] তদুপরি Mind Message পৌছে দেওয়ার অনন্যতম আরেকটি উপায় হলো Night Dreaming যখন আপনি ঘুমাতে যাওয়ার পূর্বে উপরোক্ত ধাপগুলো ফলোআপ করবেন এবং আপনার সাব-কনশিয়াস মাইন্ডে সেটআপ করে দিবেন যেন স্বপ্নের মাধ্যমে আপনার Mind Message প্রাপক অবধি পৌছে যাচ্ছে....

[বিঃদ্রঃ প্রাথমিক দিকে আপনি Brain To Brain কমিউনিকেট করতে হয়তো অসফল হবেন- যা খুবই স্বাভাবিক; তথাপি ধৈর্য ও নিষ্ঠার সাথে বারংবার প্রচেষ্টা আপনাকে নিশ্চিত সফলতা উপহার দিবে]।
